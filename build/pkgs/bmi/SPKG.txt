= BMI =

== Description ==

BMI : The BLAD Multiple Interfaces

The BLAD - MULTIPLE - INTERFACES library is a C library which implements
interfaces between the BLAD libraries, and various computer algebra systems
such as MAPLE and SAGE.

http://www.lifl.fr/~boulier/BMI
